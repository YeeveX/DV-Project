Methodology

The data used for the visualizations comes from three main sources:

  • The Armed Conflict Location & Event Data Project (ACLED) aggregated data for Asia-Pacific , which provides detailed information on political violence and protest in all of South-East Asia, Myanmar included.
  • The World Bank, which offers a wide range of economic indicators and data on various countries.
  • The Uppsala Conflict Data Program (UCDP) Georeferenced Event Dataset (GED), which provides specific and precisely geolocated data on conflict-related events in Myanmar.
